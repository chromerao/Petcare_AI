# 주제 선정 기록

> 상태: 선정 완료

## 선택 결과

- 후보 ID: T12
- 프로젝트 이름: PetCare Expert Guide
- 한 문장 문제 정의: 보호자와 시설 운영자가 반려동물 케어 문제와 운영 규정 질문을 문서 근거 우선으로 확인하고, 문서에 없는 일반 질문은 AI 조언으로 분리해 안내받도록 돕는다.
- 주요 사용자: 반려동물 보호자, 동물위탁관리업 시설 대표, 매니저, 현장 근무자, 신규 직원
- 내부 문서: 시설 운영 매뉴얼, 입·퇴실 체크리스트, 위생·격리 SOP, 사고 대응 절차, 고객 동의/보상 정책, 보호자 케어 샘플 가이드
- 외부 문서: 동물보호법·시행령·시행규칙, 국가동물보호정보시스템 영업자 안내, 소비자분쟁해결기준, 공공데이터포털 인허가 데이터, AVMA/ASPCA/FDA/AAHA 보호자 자료
- 차별화 기능: 문서 근거 답변과 AI 일반 조언을 명확히 구분하고, 독성 섭취·응급 신호·투약 질문은 안전 고지와 병원 상담으로 전환한다.

## MVP 범위

- 대상 범위: 보호자 케어 질문과 동물위탁관리업 중 반려견·반려묘 호텔/유치원 운영 질문
- 반드시 구현: 공식/내부/케어 문서 통합 검색, 문서명·조항/페이지 인용, 답변 보류, AI 일반 조언 fallback, 문서 버전/기준일 표시, 피드백
- 구현하지 않음: 수의학적 진단, 투약 지시, 약물 용량 계산, 법률 자문 결론, 실시간 예약/객실 재고 확인, 실제 진료기록 처리
- 성공 지표: Retrieval Recall@5 85% 이상, Citation Precision 90% 이상, 답변 보류 정확도 90% 이상, ACL 누출 0건
- 데모 시나리오: 초콜릿 섭취 응급 안내, 사료 변경 후 설사 관찰, 분리불안 상담, 신규 입실 확인, 탈출/상해 사고 대응, 고객 환불 문의

## 데이터 확인

- 공식 문서: 국가법령정보센터, 국가동물보호정보시스템, 공공데이터포털, AVMA, ASPCA, FDA, AAHA의 원문 URL과 기준일을 저장한다.
- 내부/케어 문서: 초기에는 `data/samples/t12_internal/`과 `data/samples/pet_owner_care/`의 합성 문서만 사용한다.
- 개인정보/민감정보: 실제 보호자 이름, 연락처, 주소, 반려동물 등록번호, 진료기록은 수집하지 않는다.
- 보존 및 삭제: 원본, 청크, 임베딩, 로그를 동일한 `document_id`로 추적하고 삭제한다.
- 외부 서비스 전송: 합성/공개 문서만 허용한다. 실제 시설 문서는 공급자 보존 정책과 계약을 검토한 후 별도 승인한다.
