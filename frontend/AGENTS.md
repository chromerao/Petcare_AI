# Frontend Agent Guide

- 프론트엔드는 FastAPI의 `/api/v1/*` 엔드포인트만 호출한다.
- OpenAI 키, 내부 문서 원문, 서버 로그, 민감 정보를 브라우저 번들에 넣지 않는다.
- 문서 기반 답변과 AI 일반 조언을 UI에서 명확히 구분한다.
- 응급/독성/투약 관련 상황은 병원 상담 안내를 항상 눈에 띄게 유지한다.
- 디자인 원본은 `DESIGN/`을 참고하되, 실제 구현은 재사용 가능한 React 컴포넌트와 타입으로 유지한다.
- API 실패, 빈 근거, 로딩, 안전 차단 상태를 각각 별도 UI로 표현한다.
